Platform

把核聚变模拟从“单模块计算”组织成“可交付工程流程”

平台围绕 IMAS 语义和标准输出目录，把每个模块的输入、参数、运行结果、图像、摘要和 manifest 都沉淀下来。它可以支撑装置方案论证、ICRH 加热方案、输运闭合、边界热负荷、诊断控制、验证 UQ 和核工程评估。

    统一入口和模块发现

    `equilibrium_state.nc`、`rf_full_wave_state.nc`、`rf_hcd_state.nc`、`core_transport_state.nc` 等文件按 IMAS IDS 语义传递。

Whole-Device Workflow

一条链路覆盖从设计点到全厂安全边界

平台模块划分参考 OMFIT/IPS 集成工作流、ITER IMAS 数据模型、whole-device core-edge 耦合和 FUSE/PROCESS 的 plasma + engineering + balance-of-plant 思路。

01

数据与装置底座

Plasma State、IMAS 索引、装置几何、机器配置和工作流 checkpoint。

02

系统设计与场景

全厂设计点、运行边界、脉冲阶段、执行器波形和工作流调度。

03

平衡 / MHD / RF

Grad-Shafranov 平衡、MHD 稳定性、ICRH 设备、full-wave 场和 RF/HCD。

04

粒子与输运闭合

结构保持 PIC、低秩张量、QL 扩散、Fokker-Planck、源汇和 1.5D core transport。

05

边界、控制与电站

边缘/SOL、壁面 PWI、诊断控制、UQ、核工程、氚循环、热工发电和安全边界。

Module Capabilities

不是把图拼在一起，而是每个环节都有可运行原型

下面这些能力来自仓库模块和 README 中的当前可运行程序，覆盖物理、工程、控制、验证和报告发布。

系统设计 / 运行场景

参数扫描、设计点、trade space、脉冲调度、运行边界和执行器时间线。

MHD 平衡与稳定性

自由边界 GS 平衡、磁面、q 剖面、MHD 稳定性和破裂逃逸电子风险。

ICRH / RF / HCD

高功率发射、匹配网络、天线耦合、full-wave RF 场、功率沉积和电流驱动。

PIC / 低秩 / 快离子

6D 结构保持 PIC、低秩张量响应、QL 扩散、Fokker-Planck 和轨道跟踪。

输运与聚变反应

中性气体、NBI/燃料、alpha 加热、中子源、杂质辐射、微湍流和 core transport。

边缘 / 壁 / 材料

台基/SOL、偏滤器热流、壁面 PWI、侵蚀、寿命限制和远程维护计划。

诊断 / 控制 / 验证

合成诊断、PID/MPC-like 执行器分配、UQ、benchmark 和低秩 vs SymPIC 对照。

核工程与电站

中子包层、氚循环、热工发电、磁体低温、材料寿命和全厂安全边界。
