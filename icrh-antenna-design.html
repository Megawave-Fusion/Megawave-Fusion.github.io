Design Modes

两类目标驱动，一套 ICRH 设计闭环

这页讲的是全流程平台中的独立产品化软件：`icrh_target_design_optimizer.py` 与新增的 `icrh_fusion_effect_design_optimizer.py`。它们让客户可以从“我要达到什么效果”开始，而不是从手工调天线参数开始。

  • ✓

    目标吸收功率 / 沉积位置驱动

    输入目标吸收功率、峰值 rho、Z 位置或电流驱动目标，扫描 strap 间距、相位、plasma gap 和频率，输出推荐天线初设。

  • ✓

    聚变效果驱动

    输入目标聚变功率、中子源、Alpha 沉积功率或 balanced-burn 目标，先运行 RF/HCD，再接入 07C 聚变反应 proxy 评分。

  • ✓

    工程约束同步评估

    候选方案同时记录 S11、VSWR、耦合电阻、strap 电压电流、表面热负荷和保护阈值，不让物理目标脱离硬件边界。

Closed Loop

从产品规格到 plasma response，再到聚变效果

软件不是孤立的表单，它会调用 ICRH 天线系统、RF/HCD、full-wave 结果、QL 源项和聚变反应 proxy，并把结果继续交给全流程平台。

01

输入设计目标

目标吸收功率、沉积 rho、电流驱动，或聚变功率、中子源、Alpha 加热目标。

02

生成天线候选

扫描产品模式、频率、前向功率、strap 宽度/间距、gap、相位和天线类型。

03

设备链路评估

计算发射机、传输线、匹配网络、耦合器、VSWR、电压电流和热负荷。

04

RF/HCD 闭环

输出功率沉积、电流驱动、径向剖面、RF/HCD map 和 `rf_hcd_state.nc`。

05

聚变效果适配

把吸收功率和沉积位置映射到温度提升、快离子增强和 07C 反应率。

06

交付结果包

输出候选排序、推荐方案、CSV、PNG、manifest 和可接入主平台的数据文件。

Real Outputs

展示软件真实生成的设计结果，而不是概念流程图

这些输出来自仓库现有运行目录，覆盖天线系统、频率扫描、RF/HCD 和聚变效果闭合。

天线与匹配系统

把发射机、传输线、匹配网络和天线耦合器放在一个工程链路里评估。

频率扫描

比较匹配、耦合和工程约束随频率变化，为产品选型和调谐策略提供依据。

RF/HCD 径向剖面

输出吸收功率、沉积位置、电流驱动和电子/离子分配，承接后续输运模拟。

聚变效果闭合

把候选天线方案按聚变功率、中子源、Alpha 加热和目标峰值位置进行排序。

Alpha / 中子输出

在设计软件里看到天线方案对聚变功率、中子源和 Alpha 沉积的影响。

反应率剖面

展示温度、密度、反应率和聚变功率分布，判断目标效果是否落在预期区域。
